``` mermaid

  NVI->>+DAPI_EWIM: findEWalletTransaction
  Note over DAPI_EWIM: Routing – Outbound – Service Provider with distributed deployment
  DAPI_EWIM->>+ESTEL: transactionsearch
  ESTEL-->>-DAPI_EWIM: transactionsearch
  DAPI_EWIM-->>-NVI: findEWalletTransaction
```

Following is a textual walk-through of the EWallet Information Management:findEWalletTransaction operation.

1.  Service Consumer invokes findEWalletTransaction operation of EWallet Information Management service to find the EWallet transactions for the provided search parameters.
2.  On receiving the request, EWallet Information Management component retrieves the transaction routing details (Endpoint URL) for the destination instance of Estel. Refer [Routing – Outbound – Service Component to route](../../appendices/Appendix G.md#routing-outbound-service-component-to-route) for details.
3.  EWallet Information Management component translates the request from CMM (Reliance SID) message to the proprietary message model of Estel and invoke the transactionsearch operation of the Merchant web service.
4.  On receiving response from Estel, EWallet Information Management component translates the message from the proprietary message model of Estel to the CMM (Reliance SID) and returns the EWallet Transaction(s) to the invoking component.
