``` mermaid

  DAPI_BRI->>+EDIF: Get
  EDIF-->>-DAPI_BRI: Get
  DAPI_BRI-->>-SC: findRecipient
```

Following is a textual walk-through of the Balance Recipient Inquiry:findRecipient operation.

1.  The consumer application (e.g. Self Care) invokes findRecipient operation of Balance Recipient Inquiry service to retrieve the list of recipients for a customer.

2.  Balance Recipient Inquiry component performs the basic semantic validations such as compliance of Recipient message with structure including the mandatory / optional elements.

3.  In case of successful validation, Balance Recipient Inquiry component translates the request from CMM (Reliance SID) to the proprietary message model of EDIF and creates the request to invoke the Get activity to retrieve the complete entries associated with provided key field(s) from an ActiveSpaces space to get the recipient list.

4.  Upon receiving the response, Balance Recipient Inquiry component translates the message from proprietary message model of EDIF to CMM (Reliance SID) and returns the list of recipients to the invoking component.
